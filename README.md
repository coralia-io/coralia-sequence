# The Coralia Sequence

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18121786.svg)](https://doi.org/10.5281/zenodo.18121786)

A 12-element integer set with Fibonacci-Lucas terminal structure.

```
C = {0, 1, 2, 3, 5, 7, 9, 12, 15, 23, 30, 35}
```

## Overview

The Coralia Sequence is generated by four arithmetic zones with ascending gaps (1, 2, 3) followed by a descending Fibonacci-Lucas cascade (8, 7, 5). It is the **unique** 12-element completion of a 10-element seed under nine axioms.

## Key Properties

| Property | Value |
|----------|-------|
| Cardinality | 12 |
| Range | [0, 35] |
| Gap sequence | (1, 1, 1, 2, 2, 2, 3, 3, 8, 7, 5) |
| Gap sum | 35 = max(C) |
| Terminal triple | (8, 7, 5) = (F₆, L₄, F₅) |

## Quick Start

```python
def generate_coralia():
    zones = [
        {'start': 0, 'gaps': [1, 1, 1]},
        {'start': 3, 'gaps': [2, 2, 2]},
        {'start': 9, 'gaps': [3, 3]},
        {'start': 15, 'gaps': [8, 7, 5]}
    ]
    result = [0]
    for zone in zones:
        current = zone['start']
        for gap in zone['gaps']:
            current += gap
            if current not in result:
                result.append(current)
    return sorted(result)
```
