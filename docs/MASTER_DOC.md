## Abstract

The Coralia Sequence C = {0, 1, 2, 3, 5, 7, 9, 12, 15, 23, 30, 35} is a 12-element integer sequence generated by four arithmetic zones with ascending gaps (1, 2, 3) followed by a descending Fibonacci-Lucas cascade (8, 7, 5).

## Zone Structure

| Zone | Start | Gaps | Elements |
|------|-------|------|----------|
| 1 | 0 | [1,1,1] | 0,1,2,3 |
| 2 | 3 | [2,2,2] | 5,7,9 |
| 3 | 9 | [3,3] | 12,15 |
| 4 | 15 | [8,7,5] | 23,30,35 |

## Axioms

- **C1:** 0 ∈ C
- **C2:** |C| ∈ C
- **C3:** |C| = 12
- **C4:** max(C) = 35
- **C5:** sum(gaps) = 35
- **C6a:** gaps[0:8] ⊆ {1,2,3}
- **C6b:** gaps[0:8] non-decreasing
- **C6c:** {1,2,3,5,7,9,15} ⊆ C
- **C7:** gaps[-3] > gaps[-2] > gaps[-1]
- **C8:** terminal gaps ∈ Fib ∪ Luc
- **C9:** terminal gaps ≥ 5
