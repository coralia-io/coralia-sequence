# Changelog

All notable changes to the Coralia Sequence project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added

- Initial public release of the Coralia Sequence
- `generate_coralia.py`: Canonical sequence generator with CLI interface
- `verify_uniqueness.py`: Verification tool for sequence uniqueness property
- `cascade_triple_enum.py`: Cascade triple enumeration utilities
- `test_axioms.py`: Comprehensive test suite for the four Coralia axioms
- `MASTER_DOC.md`: Complete mathematical documentation with proofs
- `coralia-sequence_math_v1.0.tex`: LaTeX paper for academic citation
- MIT License
- CITATION.cff for standardized citation

### Documentation

- README with quick start guide
- Mathematical definitions and theorems
- Algorithm descriptions (naive and optimized)
- Computational results for first 10,000 terms
